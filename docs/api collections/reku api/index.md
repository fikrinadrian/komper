## Welcome to Reku API

Reku API enables third-party developers and B2B partners to programmatically access Indonesia's leading cryptocurrency marketplace. Build trading bots, integrate crypto payments, or create custom trading interfaces with our comprehensive REST API.

## Base URLs

### Production

All production API requests should be made to:

```
https://api.reku.id
```

## API Features

| Feature | Description |
|---------|-------------|
| **RESTful Design** | Standard HTTP methods (GET, POST) with JSON responses |
| **Authentication** | API Key-based authentication via headers |
| **Rate Limiting** | Fair usage limits to ensure platform stability |
| **Real-time Data** | Up-to-date market prices and order book data |
| **IDR Trading Pairs** | Trade crypto assets against Indonesian Rupiah |

## Use Cases

### Trading Bots

Build automated trading strategies using market data and order placement APIs. Monitor prices, analyze trends, and execute trades programmatically.

### Portfolio Management

Create custom portfolio trackers and management tools. Fetch balances, transaction history, and calculate performance metrics.

### Payment Integration

Accept crypto payments in your application by monitoring wallet addresses and converting to IDR.

### Market Analysis

Build market analysis tools using real-time order book data, price feeds, and historical transactions.

## Getting Started

### Create Account

Sign up for a Reku account at [reku.id](https://reku.id) and complete KYC verification.

### Generate API Key

Navigate to Settings > API Keys in your dashboard to generate your API credentials.

### Make Your First Request

Follow our [Quickstart Guide](/docs/guides/quickstart) to make your first API call.
